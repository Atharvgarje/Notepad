Notepad
A simple Java-based Notepad application that provides basic text editing functionalities.

Features
Create, open, and save text files

Cut, copy, and paste text

Undo and redo actions

Find and replace text

Change font style and size

Print documents

Requirements
Java Development Kit (JDK) 8 or higher

Usage
File Menu: Create new files, open existing ones, save current work, or print documents.

Edit Menu: Perform cut, copy, paste, undo, redo, find, and replace operations.

Format Menu: Customize font style and size to enhance readability.
